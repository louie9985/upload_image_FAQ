# 图片问答上传平台 · 用户操作手册

本文面向日常使用本平台的同事，说明如何登录、批量上传图片、生成问答、审核修改并同步到企业微信智能表格。

---

## 一、平台能做什么

- 使用**系统账号和密码**登录。
- **批量上传**图片（仅支持 **PNG、JPG、JPEG**，与微信小程序兼容；请勿使用 SVG）。
- 为每张图填写**业务描述**，系统调用大模型生成**主问题、多条相似问题、答案**。
- 支持将多张图组合成**合并生成项**（例如证书正反面、多页材料），额外生成一条问答。
- 在页面中**审核、修改** AI 结果后，填写企业微信智能表格的 **Webhook 链接**，将数据上传。
- 查看**上传记录**、本账号的 **Token 用量**（大模型调用统计）。

---

## 二、登录与退出

1. 打开平台地址（由管理员提供）。
2. 在登录页输入**账号**和**密码**，点击「登录」。
3. 登录成功后，顶部右侧显示您的姓名或账号，需要离开时点击「**退出**」。

> 若忘记密码，请联系管理员在后台重置或新建账号。

---

## 三、整体流程（推荐顺序）

```text
登录 → 新建任务 → 上传图片 → 填写描述（可选合并项）→ 生成问答 → 审核修改 → 填写 Webhook → 上传
```

页面顶部有快捷按钮：**开始新任务**、**查看上传记录**。主要功能在标签页：

| 标签页     | 用途                               |
| ---------- | ---------------------------------- |
| 新建任务   | 从上传图片到生成、上传的完整工作台 |
| 历史任务   | 查看所有批量任务的状态与进度       |
| 上传记录   | 查看每次上传到企业微信的结果摘要   |
| Token 用量 | 查看本账号大模型调用累计与明细     |
| 使用手册   | 页面内嵌的简要说明（与本文一致）   |

## 四、新建任务：分步说明

### 4.1 创建任务并上传图片

1. 打开 **「新建任务」** 标签页。
2. （可选）填写**任务标题**，便于在历史任务中辨认，例如：`2026年5月证书问答批量`。
3. 点击 **「选择图片」**，可多选；仅支持 **.png / .jpg / .jpeg**。
4. 点击 **「创建任务并上传」**。

上传成功后，页面进入下一步「填写描述」。

**一键清空**：仅清空**当前页面临时工作区**（标题、已选文件、当前批次等），**不会删除**数据库里已有的历史任务或上传记录。

### 4.2 填写描述并配置生成项

每张上传的图片会对应一个**单图生成项**，默认都会参与 AI 生成。

1. 在每张图片下方的文本框中填写**描述**（**必填**）。  
   建议写清：图片业务含义、适用场景、希望问答侧重哪些点。
2. **合并生成项**（可选）：  
   - 若多张图共同表达一条知识（如证书正反面），勾选图片旁的 **「选择用于合并」**，至少选 **2 张**。  
   - 点击 **「新增合并项」**，系统会新增一张卡片，标题会自动组合所选图片名称。  
   - 在合并项卡片中填写**合并后的描述**（**必填**）。  
   - 不需要的合并项可点 **「移除」**。
3. 确认所有单图项和合并项的描述都已填写后，点击 **「保存描述并生成问答」**。

系统将弹出**进度窗口**，显示生成进度；生成过程中可关闭弹窗做其他事，任务会在后台继续。

**常见提示**

- 「每个单图项和合并项都必须填写描述」：请检查是否每一栏都有内容。
- 「请至少勾选两张图片新增合并项」：合并需要至少两张图。

### 4.3 审核和修改 AI 结果

生成完成后，页面展示每条 **问答草稿**，通常包括：

- **主问题**
- **相似问题**（多行，每行一条）
- **答案**

您可以：

- 直接修改文本；相似问题保持**一行一条**。
- 点击单条 **「保存当前修改」** 立即保存该条。

**上传到 Webhook 前**：若您修改了内容但未逐条保存，系统在您点击 **「上传到 webhook」** 时会**自动保存**当前页面上未保存的修改，再开始上传。

### 4.4 填写 Webhook 并上传

1. 在 **「Webhook 链接」** 输入框中粘贴企业微信智能表格提供的地址（一般以 `https://qyapi.weixin.qq.com/` 开头）。  
   具体获取方式由贵司管理员或企业微信文档说明。
2. 点击 **「上传到 webhook」**。
3. 等待弹窗显示上传进度直至完成。

成功后，本页下方 **「本次上传记录」** 表格会出现简要结果（问题、状态、企业微信记录 ID、图片 CDN 链接等）。

## 五、历史任务

在 **「历史任务」** 中可查看所有批次的：

- 任务 ID、标题、**状态**、图片数量、已处理数、失败数、创建时间。

用于追溯某次批量任务是否全部处理完成。

---

## 六、上传记录

在 **「上传记录」** 中可查看每次上传到企业微信后的记录，包括：

- 问题摘要、成功/失败状态  
- 企业微信侧记录标识  
- **图片 CDN 链接**（系统保存的是企业微信返回的链接，不在本地长期保存原图或 base64）

---

## 七、Token 用量

在 **「Token 用量」** 中可查看**当前登录账号**的大模型调用统计：

- 顶部 KPI：**输入 Token、输出 Token、总 Token**（累计）。
- **最近调用明细**：模型名、关联问题摘要、各 Token 数、成功/失败、时间。

便于了解个人或账号维度的 AI 消耗情况（具体配额策略由公司制定）。

## 八、常见问题（FAQ）

**Q：为什么只能传 PNG / JPG / JPEG？**  
A：与常见浏览器与企业微信侧展示兼容；本项目不支持 SVG，请勿上传 SVG。

**Q：合并项会替换单图项吗？**  
A：**不会**。合并项会**额外**生成新的问答草稿，原来的每张单图仍会各自生成一条。

**Q：生成或上传失败怎么办？**  
A：进度弹窗中会显示 **失败原因**（后端返回的错误信息）。请根据提示检查描述、网络、Webhook 地址是否正确；仍无法解决请联系管理员查看服务端日志。

**Q：数据存在哪里？**  
A：问答内容经审核后可上传到企业微信智能表格；平台侧重记录任务过程、上传结果摘要与 Token 用量，**不在上传成功后长期保存本地图片路径或 base64**（图片在企业微信侧以 CDN 链接等形式返回）。

## 九、需要协助时

- **账号、权限、密码**：联系系统管理员。  
- **Webhook 地址、智能表格字段映射**：联系对接企业微信的同事或管理员。  
- **功能异常或错误码**：向管理员提供操作时间、任务 ID、界面上的错误原文，便于排查。
